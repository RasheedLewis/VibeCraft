# Experiment Logs

Use this folder to document your experiments and findings.

## Suggested Format

Create markdown files or use the template below to track:

- Prompts that produce cool results
- Model comparisons
- Parameter combinations
- Visual notes and observations

## Example Experiment Entry

```markdown
## Experiment: [Date/Name]

**Prompt:** [Your prompt here]  
**Model:** [Model name]  
**Parameters:** 24 frames @ 8fps, 576x320, seed: 42  
**Video URL:** [link]  
**Video Duration:** 3.0s (calculated from frames/fps)  
**API Call Duration:** 45.2s  
**Observations:** [Your notes about quality, style, what worked/didn't work]
```
