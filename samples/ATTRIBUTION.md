# Sample Audio Attribution

Track attribution requirements for downloaded sample audio files.

## Format

For each song, add an entry with:
- **Filename:** The audio file name
- **Attribution:** Full attribution text (what will be displayed)

---

## Songs

### `its-not-hard-to-get-lost.mp3`
**Attribution:** "It's Not Hard To Get Lost" by Bryan Mathys - Free Music Archive - CC BY 4.0

---

## Notes

- If no attribution is listed, the song doesn't require attribution (e.g., CC0, Pixabay License)
- When uploading to the app, copy the attribution text into the `attribution` field
- Keep this file updated as you add new sample songs
